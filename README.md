# GreenGuide AI

## Chosen Vertical

Sustainability / Carbon Footprint Awareness

## Project Idea

GreenGuide AI is a smart assistant that helps users understand and reduce their daily carbon footprint. It gives practical eco-friendly suggestions based on user habits such as travel, electricity usage, food choices, and waste management.

## Approach and Logic

The assistant asks the user simple questions about their daily lifestyle. Based on the answers, it identifies high-impact habits and suggests better alternatives.

Example:

* If the user travels by bike or car for short distances, it suggests walking, cycling, or public transport.
* If the user wastes electricity, it suggests switching off unused devices.
* If the user uses plastic often, it suggests reusable alternatives.

## How It Works

1. User enters their daily activity.
2. The assistant analyzes the activity.
3. It gives a carbon awareness score.
4. It suggests eco-friendly actions.
5. It motivates the user with simple daily goals.

## Assumptions

* The app gives awareness-based suggestions, not exact scientific carbon calculations.
* The goal is to encourage small daily actions for a greener future.
